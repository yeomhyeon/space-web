# ✦ STELLARIA — 감정을 채집하는 우주

광활한 3D 우주를 유영하며 별처럼 떠다니는 이모지를 채집하고, 모은 감정을 조합해
세상에 하나뿐인 폭죽을 만들어 밤하늘을 밝히는 인터랙티브 웹 경험.

**탐험 → 채집 → 창작 → 발사 → 기록** 의 흐름으로 이어집니다.

## 실행 방법

빌드 과정이 필요 없습니다. 로컬 정적 서버만 띄우면 됩니다 (ES 모듈/웹캠 때문에 `file://` 직접 열기는 불가).

```bash
cd "우주 웹"
python3 -m http.server 8000
# 브라우저에서 http://localhost:8000 접속
```

Three.js / MediaPipe 는 CDN(jsdelivr)에서 자동 로드되므로 인터넷 연결이 필요합니다.

## 조작

| 동작 | 데스크톱 | 손동작(웹캠) |
|------|----------|----------------|
| 확대·축소 | 마우스 휠 | **양손 핀치(위치형)** — 벌린 채 유지하면 계속 확대, 모은 채 유지하면 계속 축소, 중립 간격으로 돌아오면 멈춤 |
| 시선 이동 | 드래그 | 펼친 손을 **가장자리로 밀기** — 손이 가는 쪽을 바라봄(1인칭 룩, 손 오른쪽→오른쪽을 봄) |
| 전진·후진 유영 | `W A S D` (상승/하강 `E` `Q`, 가속 `Shift`) | 손바닥을 **앞으로 밀기 / 몸쪽으로 당기기** |
| 채집 | 이모지 위 클릭(짧은 탭) | 손 커서를 이모지에 두고 **주먹** |
| 발사 | `Space` 또는 미리보기의 발사 버튼 | ✊ **주먹 꾹 쥐고 ~1초 유지(충전)한 뒤 손을 펴서** 발사 |
| 정보 보기 | (호버) | 이모지 위에서 **손 멈추기** |

손동작 우주 조작은 인트로의 "✋ 손으로 우주 탐험하기" 또는 우상단 ✋ 버튼으로 활성화합니다.
손을 움직이면 **화면에 손 커서(레티클)** 가 따라다녀 어디를 가리키는지 보이고, 이모지에 맞추면
커서가 핑크빛으로 강조되며 어떤 별을 채집할지 라벨로 알려줍니다(그 상태에서 주먹을 쥐면 채집).
마우스·키보드 조작은 항상 함께 유지되며, 웹캠이 거부되거나 미지원이면 자동으로 폴백합니다.
민감도/방향은 `universe.js`의 `handYawSpeed` 등 4개 상수로 조정할 수 있습니다.

## 핵심 시스템

- **3D 우주** — 3겹 패럴랙스 스타필드 + 성운 + 오로라, 부드러운 카메라(OrbitControls + 유영), UnrealBloom 네온 글로우
- **이모지 = 폭죽 속성** — 16종의 이모지가 각각 형태·색감·반짝임·잔상·속도·꽃잎·꼬리 등 하나의 특성을 담당 (`src/data.js`)
- **폭죽 제작** — 3~5개 조합. 시드 + 약간의 랜덤 변주로 같은 재료도 매번 조금씩 다른 결과. AI풍 이름·희귀도·효과 설명 자동 생성 (`src/craft.js`)
- **폭죽 연출** — 대형 이모지 폭죽 (`src/fireworks.js`)
  - **입자 = 실제 이모지**: 조합 이모지로 텍스처 아틀라스를 만들고, 단일 `Points` + 커스텀 GLSL 셰이더로 폭죽당 **수천 개**의 아주 작은 이모지 입자를 한 번의 draw call에 렌더 (위치는 셰이더에서 `uTime`으로 계산 → GPU 시뮬레이션)
  - **형상 완성**: 입자들이 중심에서 목표 형상으로 퍼져 모양을 만든다 — ❤️=하트, ⭐=별 스파이크, 🌸=5장 꽃잎, 🌙=초승달, 💎=은하, ⚡=날카로운 직선 빛줄기 등 조합별로 색·형태가 확연히 달라짐
  - **발사 시점 정면 정렬**: 터질 때 형상의 정면이 **발사 순간 카메라가 바라본 방향**을 향하도록 회전(쿼터니언). 어느 각도에서 쏴도 정면이 보이고, 터진 뒤엔 시점을 돌려 360° 다양한 각도로 감상 가능
  - **다단계**: 상승(혜성 꼬리) → 중심 섬광 → 방사형 빛줄기(`LineSegments`) → 형상 폭발 → 2차 폭발들 → 모든 입자가 완전히 페이드아웃 (터진 자리에 불꽃 잔상을 남기지 않음)
  - 멀리서는 화면을 채우는 웅장한 폭죽, 가까이 줌인하면 작은 이모지들이 촘촘히 모여 있는 게 보임 (point size attenuation)

- **우주 무드** (`src/universe.js`) — 실사 딥스페이스: 거의 순수한 블랙 배경에 **오밀조밀한 미세 별 약 4.6만 개**(3개 패럴랙스 레이어 + 은하수 띠), 색온도 편차(청백색 다수·주황/적색 소수), **회절 광채가 있는 밝은 주요 별**, **은하수 먼지띠와 성단 4곳**, 절제된 딥블루 먼지구름. 만화 같은 컬러 성운/오로라 없이 모던·웅장·감각적인 톤.
- **우주 진화** — 발사할수록 별·성운·오로라가 늘고 블룸이 강해짐
- **도감** — 새 조합 자동 등록, 이름/희귀도/이모지/제작 횟수/첫 발견일/다시보기, 완성률(`/250`). `localStorage`에 영속 저장

## 파일 구조

```
index.html        진입점 · UI 오버레이
styles.css        글래스모피즘 · 네온 디자인
src/
  main.js         오케스트레이터 (화면 전환·이벤트·렌더 루프)
  universe.js     Three.js 씬 · 스타필드 · 이모지 · 진화 · 블룸
  fireworks.js    발사 · 폭발 파티클 시스템
  craft.js        레시피 생성 · 이름 · 희귀도 · 속성 집계
  hands.js        손동작 우주 조작 (MediaPipe Tasks Vision · HandLandmarker · 지연 로드)
  data.js         이모지 속성 데이터 · 이름 사전
  store.js        컬렉션 · 도감 · 발사 영속 상태
```

## 기술

Three.js (ESM, importmap), OrbitControls, EffectComposer + UnrealBloomPass,
MediaPipe Tasks Vision (HandLandmarker, 양손), Canvas 2D 이모지 텍스처, localStorage. 번들러 없이 순수 ES 모듈.
